# 🚀 GuerrillaTech School Management System

## 🌟 Project Overview
An innovative, ethical, and comprehensive school management platform designed to integrate STEM education, AI innovation, and entrepreneurship.

## 🏛️ Organizational Structure
The system is organized into five key strategic divisions:
- **Admiralty**: Strategic command and high-level governance
- **Command Division**: Operational coordination and partnerships
- **Ohio Tech Academy**: STEM education and apprenticeship management
- **SallirreugTech**: Innovation and product commercialization
- **Research and Ethics**: AI research and ethical framework development

## 💡 Key Features
- Responsive, modern UI with interactive dashboards
- Division-specific metrics and management tools
- Strategic initiative tracking
- Real-time organizational insights
- Ethical AI integration
- Comprehensive reporting and analytics

## 🛠️ Technology Stack
- **Frontend**: Next.js 14
- **Backend**: Prisma ORM
- **Authentication**: Clerk
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL
- **Optional**: Cloudinary for media management

## 🚀 Quick Start

### Prerequisites
- Node.js (v18+)
- PostgreSQL
- Clerk Account
- (Optional) Cloudinary Account

### Installation Steps
1. Clone the repository
\`\`\`bash
git clone https://github.com/yourusername/full-stack-school.git
cd full-stack-school
\`\`\`

2. Environment Configuration
- Copy \`.env.example\` to \`.env\`
- Fill in Clerk Authentication Keys
- Configure Database Connection URL
- (Optional) Add Cloudinary Credentials

3. Install Dependencies
\`\`\`bash
npm install
\`\`\`

4. Database Setup
\`\`\`bash
# Run database migrations
npx prisma migrate dev

# Seed initial data (optional)
npm run prisma:seed
\`\`\`

5. Start Development Server
\`\`\`bash
npm run dev
\`\`\`
